# DecodeLabs Data Analytics Project 1

## Objective

Clean a raw dataset by:

- Handling missing values
- Removing duplicates
- Correcting data formats

## Tools Used

- Python
- Pandas
- Jupyter Notebook

## Tasks Performed

### Missing Values
Filled missing CouponCode values with "No Coupon"

### Duplicates
Checked and removed duplicate rows

### Date Formatting
Converted dates to YYYY-MM-DD format

### Text Cleaning
Removed extra spaces from text columns

### Numeric Formatting
Rounded numeric columns to 2 decimal places

## Results

- Missing Values = 0
- Duplicate Rows = 0
- Duplicate Order IDs = 0
