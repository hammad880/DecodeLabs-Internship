# DecodeLabs Data Analytics Internship Tasks

## Overview

This repository contains all completed projects from the DecodeLabs Data Analytics Internship Program. The projects cover the complete data analytics workflow, starting from data cleaning and preparation to exploratory analysis, SQL querying, and data visualization.

# Project 1: Data Cleaning & Preparation

## Objective

Clean a raw dataset by identifying and handling:

- Missing values
- Duplicate records
- Incorrect data formats

## Tasks Performed

- Identified null values
- Handled missing data
- Checked and removed duplicate records
- Standardized date formats
- Cleaned text fields
- Validated dataset integrity

## Skills Learned

- Data Cleaning
- Data Preparation
- Data Quality Assessment
- Pandas Fundamentals

## Tools Used

- Python
- Pandas
- Jupyter Notebook
